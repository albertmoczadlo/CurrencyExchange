# Vistula Exchange

Vistula Exchange is my personal online currency exchange project built with ASP.NET Core and Razor.

## Scope

The application includes:

- an admin area for exchange rates, reports, money stock, and user management
- a user area for deposits, withdrawals, currency exchange, transaction history, and price alerts

## Stack

- ASP.NET Core
- Entity Framework Core
- Razor Views
- SQL Server
- Bootstrap

## Current Focus

The current work is focused on:

- stabilizing the architecture and asynchronous flows
- upgrading the solution from .NET 6 to a supported runtime
- cleaning warnings and preparing the app for a full visual redesign
